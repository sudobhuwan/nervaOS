## 📝 **Technical Details:**

**Parser:** Custom markdown parser in `message_renderers.py`
- Handles: Headers, Lists, Code, Links, Bold
- No external dependencies
- Clean, semantic HTML-like structure

**Renderer:** GTK4 widgets with CSS
- Gradient backgrounds
- Custom fonts (JetBrains Mono for code)
- Smooth transitions
- Hover effects

**Performance:** Fast & smooth
- Lazy rendering
- Efficient parsing
- GPU-accelerated animations
